# University Labs

Университетские лабораторные работы.

## Структура

| Папка | Дисциплина | Язык / формат |
|-------|-----------|---------------|
| [`ASD`](ASD)   | Алгоритмы и структуры данных | Python |
| [`C++`](C++)   | Программирование на C++ (в т.ч. SFML) | C++ |
| [`DM`](DM)     | Дискретная математика | Python |
| [`OOP`](OOP)   | Объектно-ориентированное программирование | Python |
| [`OS`](OS)     | Операционные системы | Отчёты (.docx) |
| [`Python`](Python) | Программирование на Python | Python |

## Примечания

Скомпилированные файлы (`.exe`, `.dll`, `.lib`), папки сборки (`build/`) и библиотеки
SFML исключены из репозитория — см. [`.gitignore`](.gitignore). В репозитории хранятся
только исходные тексты, ресурсы и отчёты.
